今日の顔(#^.^#)

最近、「顔」みたいなものが妙に気になります

周りに言ったら、全然気にならないそうです（笑）



先日、お店で出てきたサラダはこんな「顔」していました。

これ絶対、店員突っ込んで欲しいんやろ！？

と思ったけど、店員さんは普通でした(^_^;)



なんか小さな発見で癒される
今日この頃でした☆


みなさんも面白い「顔」を
見つけたら教えてください(#^.^#)

税金が心配です。。。

大林　由衣 (2011年10月24日 11:36) 0 0 票 0 票


震災の復興財源として、

消費税の引き上げが確実になりました。。


2015年までに消費税を10％にまで引き上げる動きです。


これでなんと、一般家庭で、


年間15万円の負担増となってしまいます！

（年収450　から　550万円の場合）




さらに、震災復興税として、

10年間の所得税が一律+4%の引き上げされたり、


たばこ税の1本1円引き上げと


...家計は苦しくなるばかり...




その、これからどんどん増えていく出費を


どうにかして削減したい！！




でもどうやって？




...実は、ガスや電気の契約を、


家族のライフスタイルに合わせて


組み替えるだけで



光熱費の額を大幅に削減することができます！





なんとうちの社員は実際に月5万円近く削減できました！





（光熱費が　プラス　33656円→　マイナス　13817円に　

その差47473円）


光熱費がマイナス？ってなんだ？







ライフスタイルに合わせて、


これは、それぞれの家族構成、

いろいろな方法がありますので、

度気軽にお声掛けください。



あなたにぴったりの光熱費削減法を一緒に考えます！


詳しくは

「光熱費アドバイザー　落合」まで


「光熱費削減について...」とお問い合わせください


メール：ochiai@e-shinwa.net

でんわ：072-252-1049



方針発表会無事終了

大林　由衣 (2011年10月17日 11:25) 0 0 票 0 票
10月15日の方針発表会、
無事終了しました。

みなさんありがとうございました。


当日は厳かな雰囲気の中、
社長をはじめ、
各部のリーダーが方針と想いを語り、

今期の目標に向け
意気込みを新たにしました！


二部の懇親会では、取引業者さんや
来賓の方も一緒に、景品をかけた
ゲームで盛り上がりました！


来賓の方々にも
コメントを頂き、とってもためになりました。

来年もこうやって皆で方針発表会を楽しんでできるように
1年がんばっていきましょう！

ちなみに、私は
ケーキの早食いに挑戦しましたが
景品にはいたらなかったです。
しかし、じゃんけん大会では
一等賞！！(*^_^*)
高そうな大吟醸をいただきました！
お父さんへのお土産にしよ☆

休日☆

大林　由衣 (2011年10月11日 23:12) 0 0 票 0 票

昨日は友達の出産祝いに行ってまいりました。

 

出産祝いといってもだいぶ時間がたってしまっているので

友達二人の子供は10ヶ月と5ヶ月になっていました。

もっと早くお祝いしたかったんですが...

 

みんなで集まって、ずっと子供たちを見ながらおしゃべりしていると、

飽きなくて、あっというまに時間がたってしまっていました。

 

なんだかゆったりとした時間が流れていて、

癒されました☆

 

お友達、ママになっても見た目は短大生のころのまま変わらないのに、

生活はすっかり変わっていましたね(*^_^*)

 

その日は、夜からはまた別の短大の友達の結婚式の二次会に出席。

 

みんなどんどん大人になっていくなぁと感じました。
