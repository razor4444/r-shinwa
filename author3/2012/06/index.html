大林　由衣: 2012年6月アーカイブ

子育てにいい！

大林　由衣 (2012年6月22日 22:44) 0 0 票 0 票
最近じめじめと雨が続きますね?

でも、雨が続くと、晴れの日が何倍も嬉しく感じる！

雨が続くと、庭の花木が元気になって新しい芽が出てる！


梅雨は植物にとって大切な季節なので、一概に嫌がれませんね（＾ｖ＾）



本日23（土）予定していた
「二世帯住宅勉強会」ですが、

日にちを絞らずに、
随時開催していこうと思っています。


平日は
1.　11：00　から
2.　14：00　から
3.　19：00　から

土・日・祝は

14：00　から


いつでも来れる日にご予約ください（＾ｖ＾）/

二世帯に限らず、家づくりに大切なこと全般についてお話と
質問なんでもお受けします♪

お気軽にどうぞ（●＾o＾●）










話は変わって、
最近気になるものがあります。

「横峰式教育法」
です。


エチカの鏡で紹介されブームになったらしいんですが
今更知りました。

「子供は誰でも天才だ」
「子供の持っている力を引き出すだけ」

それだけで
4歳で絶対音感が付き
5歳で小2レベルの読み書き計算ができるようになり
6歳で逆立ち歩きができるようになり

などなど


これを、子どもに強制するのではなく
楽しんで出来るようになるのなら、すごいなぁと思います！！


今とっても気になります(・o・)/


横峰式英才教育
詳しくはこちら↓
http://www.yokomineshiki.jp/



昔の知恵に学ぶ

大林　由衣 (2012年6月 5日 15:55) 0 0 票 0 票
前回のブログに書きました
二世帯住宅の勉強会ですが

日程を変更して

6/23（土）になりそうです。

確定しましたら、御案内します(*^。^*)

二世帯住宅だけでなく、
どんな家にも必要な考え方をお話しますので

どなたもぜひおこしください（＾ｖ＾）
これを知らずに家を建てると損してしまうと
思います。。。！

家づくりは一生のことなので
しっかり知識を入れて、最高の我が家を
つくってくださいね（＾ｖ＾）/





私は先日、伊賀の忍者村に行ってきました。

そこには「忍者屋敷」の復元があって、

いろんな室内の仕掛けを
ガイドの方が教えてくれます☆


その仕掛けがとても面白く、

今の家づくりにも活かせるのではないか
昔の知恵ってすごい！！

と感動しました。


例えば、大事な書類を隠す床下収納や
刀を隠す、床板や
暗さを利用した、中からは見えるけど
外から見えない見張りスペースや、
一見棚に見える隠し階段、
壁のどんでん返しで各部屋に繋がる隠し通路、
簡単に素早く開く木の鍵など

これをそのまま活かしても「スパイの家」になりそうですが(^_^;)

最新の機械や設備に頼らなくても
こんなにすごいスパイ（忍者）屋敷が作れるんだなと
勉強になることばかりでした。

私は忍者はなんだか迷信っぽく、本当に実在した
イメージが今までなかったのですが

本当にいたんだなと思い、すごく興味がわきました。

そしてまだまだ知らない事がたくさんあるなと感じました。
知らない事を知れたいい機会でした☆

小さなお子さんは忍者の衣装を着て走り回っていて
とてもかわいかったです(*^。^*)

ぜひご家族で行ってみてください♪

近くに有名なアスレチックもあって、ご家族にお勧めです☆

伊賀忍者村
http://www.iganinja.jp/

フォレストアドベンチャ―　山添　
http://www.forest-ad.jp/adventure.html
