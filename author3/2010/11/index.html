寒さに負けないように

大林　由衣 (2010年11月29日 08:41) 0 0 票 0 票

最近寒くなりましたね。

 

寒くなると、血行が悪くなるから、私は肩こりがひどくなる季節です(>_<)

 

先週からひどい肩こりで、頭痛と吐き気がして...

昨日やっと鍼灸院に行ってきました。

 

友達がやっていて、保険もきいて安いので、時間があれば行くように

していますが、最近は時間がなくて、3ヶ月ぶりくらいに行きました。

 

針と電気と指圧で1回でだいぶ回復しました。

これだけしても７００円弱でできます☆

 

やっぱり暖めることが大事ということで、お風呂も首までしっかりつかったり

寝るときも首を冷やさないように気をつけるのが大事らしいです。

 

そこでお勧めなのが、ジェル式でレンジで暖めてつかう、柔らかい湯たんぽです。

これを寝るとき、足元ではなく、肩に敷いて寝ると、だいぶ楽になりますよ☆

 

 

友達は今年いっぱいで産休に入るので、なんとか今年のうちに

あと数回顔を出したいところ...

 

そして産休で暇になるのを心配していたので、何か家でゆっくりできる趣味の情報を

持って行こうかなと思っています。

 

何がいいですかね(^_^)

編み物とか...？何か作ったり、手芸とか、料理とか

考えるといろいろありそうで、楽しそうですね♪

いろいろ探してみようっと(#^.^#)

 

デザインとエコの融合♪

大林　由衣 (2010年11月24日 09:38) 0 0 票 0 票
先日11/21（日）住宅の完成見学会を行いました。

1日の開催でしたが、たくさんの方が見に来てくださいました。

今回はもうお住まいになられている状態を
お施主様がこの見学会のために開放していただきました。

そして、見学会中も説明をしていただいて、
お施主様には大変ご協力をいただきました。
お忙しい中、本当にありがとう御座いました。


見に来てくださったお客様も、おうちの素敵さに
とても驚いていました。

家具もおうちに合わせ一流の家具をトータルにコーディネート
されているので、なかなか建売や一般のモデルハウスでは
見られない光景だったと思います。

ぜひこのお家のなかで、
「ここいいな、こんなとこ我が家に活かそう」と
思っていただけたらと思いました。

そして、この素敵なデザインはもちろんのこと、
隠れたところにたくさんの工夫があり
究極のエコハウスになっています。
まさにデザインと機能の融合ですね。


これから建てる家は次世代のことを考えて、
資産価値を持続できることはもちろん、
エコに配慮し、CO2を出さない家をつくり、
次世代に安心な環境を残して行きたいですね(#^.^#)


この見学会の様子を、見に来てくださったお客様が
自身のブログに詳しく載せて下しました☆
ありがとう御座います☆

ぜひそちらもご覧下さい♪
ブログ↓
http://hcasa.exblog.jp/13690466/


当日お忙しい中見に来ていただいたみなさん
本当にありがとう御座いました！！

ぜひ御参考に、ご自分のおうちのイメージを広げていってください♪(#^.^#)

Happy　♪

大林　由衣 (2010年11月15日 10:30) 0 0 票 0 票

昨日は大親友の結婚式でした。

 

短大で知り合った友達ですが、数多くの友達の中、

短大からは一人だけ席を用意してくれて、

幼馴染のお友達に混じって挙式から出席させていただきました。

 

短大の友達みんなの代表という気持ちで、

みんなに見せなきゃと思い、いっぱい写真撮りました。

 

ほんとうに芸能人のようにきれいな子なので、絵になる！

お姫様みたいでしたー(#^.^#)

 

みんながサプライズでビデオレターをいっぱい用意していて、

新郎新婦の人柄の良さからの、人望だなぁと感じました。

 

そして、ブーケのプレゼントでは（最近はトスではなく、リボンを引くくじ引きなんですね）

みんなが、これだと思うリボンを選んでる中、無心で選ぶと...

 

 

なんと大当たり！

 

ブーケ貰っちゃいました☆

 

なんか幸せ分けてもらえた感じがして、すごく嬉しかったです♪(｡^_^｡)

 

 

二次会では短大メンバーでサプライズの歌のプレゼント♪

 

ＡＩ　のstory　っていう曲をの歌詞を替え歌にしてみんなで歌いました。

何度も集まって練習し、思い出の画像に合わせて、歌詞が出るカラオケ画像のような

ＤＶＤまで作成して、すごく頑張りました☆

 

なんとか泣かずに全部歌えたし、集まって練習することで、友達の絆も深まり、

とても良かったです。

 

私もこんなにみんなに祝福してもらえるような人間にならないとなと思いました♪

 

Ｍ夫妻本当におめでとう御座います☆

 

 

 

 

 

 

 

 

 

 

 

 

ついに決定♪

大林　由衣 (2010年11月11日 16:06) 0 0 票 0 票

以前から準備しておりました、

エコ住宅の完成見学会がついに開催できることになりました！

 

☆・:*:・☆ ・:*:・・☆ ・::*:・☆・:*☆・:*:・

 

場所は　堺市南区御池台（泉北ニュータウン）です。

日にちは11/21（日）10：00から17：00です。

 

 

ここはカーサソーレではないのですが、コンセプトはカーサソーレと同じ

「エコ」の家です。

 

何がエコかというと、

1.       エコポイント基準の断熱材とlow-eガラスを使用　　

2.       京セラの太陽光パネルを搭載　　　　　　　　　　　

3.       ガスの省エネ給湯システムのエネファーム　　　　　

4.       珪藻土の呼吸する塗り壁　　　　　　　　　　　　　　　

5.       無垢の床材　　　　　　　　　　　　　　　　　　　　　　　

6.       ＬＥＤ照明　　　　　　　　　　　　　　　　　　　　　　　

7.　   炭で消臭

8.        地下室の外断熱

 

など、ソーレと共通したり、比べたりできるポイントがたくさんあります。

施主様こだわりの最先端の省エネ住宅です。

このようなエコにこだわった住宅の完成見学会の機会はめったにないので、

ぜひ一度、実際に見て、室内に入って、住み心地を体験してみてください。

 

 

家具も全て入って居ますので、ただの完成したがらんとした家ではなく、

これぐらいの大きさの家具が入ると、こんな風になって、

こんな空間になるんだなということがすごくよく分かります。

 

また、住んでもらった上でのリアルな感想も届いています。

詳しくはブログで↓

http://www.r-shinwa.jp/kodawari/2010/10/post-34.html

 

 

よろしければ、一度実際に中に入って快適さを感じてください♪

見学会はいろいろな家具が入っていますので、

込み合わないように予約制にさしていただきたいと思いますので

行ける時間があればご一報ください♪

子どもさんとご一緒も歓迎です。

11/21（日）

10：00から17：00

です。

 

地図などの詳細はお申し込み後お知らせ致します♪

営業などは一切御座いませんので、休日のイベントとして御気軽にどうぞ☆ 

知らなきゃ損する補助金・減税の話

大林　由衣 (2010年11月 5日 19:42) 0 0 票 0 票

今年は住宅ローン減税、贈与税減税、長期優良住宅の金利優遇など、

数々の優遇処置が取られ、住宅が購入しやすい政策がされています。

 

また、補助金も、太陽光補助金や長期優良住宅、住宅版エコポイントなど、

今年が最大のメリットとなっています。

 

そして、一部減税や補助金は今年で終わってしまったり、来年まで延長されていたり、

なかなかややこしいものです。

一人で調べようとしてもめんどくさいし、さらに、補助金は予算がなくなれば、そこで終了したり、

また逆に、予算が余ったり追加されれば来年まで延長されたりと、

 

最新の情報を追うのも一苦労なのです。

 

でも知らないまま、いつの間にか終わってしまったり、利用しなかったりすると

何百万も損することになります！！本当です(>_<)

 

 そうならないように、

この補助金やローン減税について、丸一日かけて一覧にまとめました！！

...けっこうたいへんでした(^_^;)

 

この一覧表、基本的にcasa soleに対応するものを集めているのですが、

他の住宅にも対応するものももちろんあります。

 

自分でいうのもなんですが、なかなか役に立ちますよ♪(#^.^#)

 

なのでぜひプレゼントしたいと思います。

以下に「補助金　資料希望」とメールください♪

shinwahome@casasole.jp

 

 

また、これをみても、

「何でこうなるの？」

「これはどんな住宅に対応するの？」

「自分の年収だとどれぐらいの効果があるの？」

とたくさん疑問が出てくると思います。

 

そんな疑問を集めて、一気に解決できるような勉強会もしていきたいと

思いますので、どうぞその機会は、どなたでも無料でお気軽にご参加ください♪(#^.^#)
