年末

大林　由衣 (2010年12月28日 19:04) 0 0 票 0 票

年末ですね。

皆様バタバタしているのではないでしょうか(#^.^#)

 

今日が仕事納めの方が多いと思います。

 

私は明日までです☆

 

明日は大掃除です！

一日で片付くか心配です...(^_^.)

 

一年あっという間ですね。ついこの間

大掃除していたような気がします。

 

こうやってあっという間に年月が過ぎて行ったらどうしよう！

と急に怖くなりました！

ちゃんと仕事だけじゃなく、人生にも目的、目標を持って

生きていかないと！

 

お正月は実家でゆっくりしながらも、人生の目的、目標について

ゆっくり考え直そうと思います。

 

あと、ずっと読みたくて溜まっている本を読み倒したいと思います(^_^)/

 

皆様、お仕事によっては出勤の方もいらっしゃると思いますが、

 

お休みの方は、ぜひゆっくりとお正月、家族と過ごて疲れを癒し、

普段したくてもできていなかったことなどを片付ける有意義なお休み

なりますように(#^.^#)

 

良いお年を(^_^)/

クリスマスです♪

大林　由衣 (2010年12月24日 10:50) 0 0 票 0 票

今日と明日はクリスマスですね♪

 

只今、本社がこんな感じにイルミネーションされています。

例年よりもとってもパワーアップ

していて、まるで

メルヘンの世界みたいです！

 

 

日本ではこんな感じで

クリスマスをしていますが、本

場イギリスなどは全然違うみたいですね。

 

イギリスの小説で、

 

クリスマスは年賀状のようにクリスマスカードを贈って、

いろいろなパーティへいく。

そして知人にプレゼントを渡す。

 

とありましたが、

このプレゼントがくせ者で、

 

自分が用意していなかった相手から突然貰うと申し訳なくなり、

逆もまた気まずく...

 

プレゼントの内容もすごく悩み、高価すぎたり、

チープすぎたりして相手のものと吊り合わなくても気まずくなり、

 

親戚などからはいらないものばかり頂き...

と、本当に大変で、

 

「こんなにプレゼントでわずらわしい思いを毎年するくらいなら、

非キリスト教圏でピカピカして楽しいクリスマスを過ごしたい！」

というせ台詞があり

 

それ、日本だ(#^.^#)と思いました。

 

よく、本当のキリスト教のお祝いとはかけ離れていると言われ

批判されてるのかと思いきや、

本場イギリス人からそんなふうに思われるなんて、

面白いなと思いました(#^.^#)

 

私も今までは、「クリスマスは子どもがサンタさんからプレゼントを貰う日」

という印象が強かったんですが、これからは

「日頃お世話になっている人達に感謝のプレゼントを贈る日」にしたいと思います。

 

みんなで勉強会♪

大林　由衣 (2010年12月20日 17:53) 0 0 票 0 票

先日12/19（日） 住宅の勉強会を開きました。

 

少人数制の座談会形式で、

みなさん初対面同士でしたが

ざっくばらんに楽しく話し合いました(#^.^#)

 

まず、家づくりの基本知識といたしまして

「家族の一生を考えるいえづくり」をテーマに

社員の実体験をもとに失敗事例、注意点などを話し合いました。

家づくり、甘く見ていると、大変な落とし穴がありますよ！

みなさんこれを聞いて参考に、気をつけてください(^_^)

 

そして次に資金計画のお話で

「失敗しないための資金計画」というテーマで

 

住宅資金計画のプロフェッショナル、

 

500組以上の資金計画をお手伝いしている

FP（ファイナンシャルプランナー）の宮田先生をお招きして

専門的なお話をしていただきました。

 

（宮田先生はとても有名な方で、

アメブロではFPブログランキング第一位なんですよ♪）

 

 

資金計画と聞くと一見難しそうですが、

質問形式でゆっくり進めてくれたので

とても分かりやすく勉強になりました。

 

 

・退職金制度がどんどん変わっている

・これからの法律、何もしなくても月1万円が消えていく！？

・●●をすると600万円も得をする！！

・どんな金融商品よりも効率がいいものとは！？

 

などなど、

 

へ?！ほ?！という感じでした(#^.^#)

 

一つ賢くなれた気がします☆

 

 

 

そして、住宅購入はただ漠然と、素人の判断やうわさで進めてしまっては

危険だなぁということをしみじみ感じました(^_^；)

住宅は、物もお金も一生ものですからね...(>_<)

 

 

今の資金振りだけではなく、一生分の計画を立て、安心して

計画通り暮らしていけるようにしたいですね☆

 

 

そのお手伝いをしますので、いつでも相談ください♪

 

皆さんの夢を無理なく、楽しくかなえられるように

願っています！(#^.^#)/

 

今回年末でお忙しい方も多かったと思いますので、

また年明け開催したいとおもいます☆

 

今回来られなかった方は、ぜひぜひ次回遊びに来てください♪

 

クリスマスプレゼントはお決まりですか？

大林　由衣 (2010年12月13日 13:57) 0 0 票 0 票
       

あなたのための「ライフプラン」を作成いたします。

大林　由衣 (2010年12月13日 11:20) 0 0 票 0 票

住宅を考える際、最初に知っておきたいのは

自分がいくら借りられて、どれくらいのローンが組めるのかです。

 

今はネットで簡単にシミュレーションできますが、

借りられる額と、借りていい額は違います。

 

そこで、自分の判断で、間違った認識をしないように、

プロに相談することをお勧めします。

 

プロのFP（ファイナンシャル・プランナー）に相談することで、

今借りられる額だけでなく、一生涯に渡っての収入の波や、

学費が多く必要になる時期など、

一生涯における収入と支出を

綿密に計算し、グラフに表すことができます。

 

それにより、生涯でローンに困ることなく

安定した暮らしができるよう、人生の設計図を作ることができます。

これが「ライフプラン」です。

 

また、住宅会社とは別の独立したFPに相談することで

公正・中立な立場で、的確なアドバイスを受けることができます。

 

しかしこの、「ライフプラン」の作成は個人で申し込むと

5万円相当かかってしまいます...。。

 

それをなんとか手ごろに活用して、役立てていただきたいと思い、

今回12/19（日）に堺で資金計画の勉強会を開きますので、

そちらに来てくださった方には、特別に無料で「ライフプラン」を作成いたします！

 

ぜひぜひこの機会を逃すことなく、活用ください♪

あなたには家づくり失敗して欲しくないから...

餅つきだょ♪

早いもので、もう今日から12月に入りましたね！



12月といえば、忘年会やら大掃除やら、里帰りやら、御節準備やらで

皆さん大変お忙しい時期だと思います！
