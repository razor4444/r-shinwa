2012年2月アーカイブ

リニューアル☆

大林　由衣 (2012年2月17日 14:48) 0 0 票 0 票
進和ホームの親会社、進和建設工業のＨＰがリニューアルしました。

見やすくなったと思うので、また、のぞいてみてください♪

http://www.e-shinwa.net/

デザインが全然違うので、
「間違えた！？」と思わないでください(笑)



進和建設工業株式会社をまだ知らない方もいらっしゃると思います。

進和建設工業株式会社は
賃貸マンションの施工をメインにしている会社で
年間マンション施工棟数が、

近畿エリアでなんと3位！（＾ｖ＾）





























自己資本が23億もあり、

税金を正しく納めた証しである

「優良法人」の表彰を3度も受けています。

堅実かつ、自己資本を厚く、
そしてなんと建設業では凄く珍しい
無借金経営！！！













だから「絶対につぶれない会社です」(*^_^*)






だから、進和ホームで家を建てても大丈夫、

何十年後にメンテナンスをしたくても
会社がつぶれてもうない！という心配がありません！


それこそ、工務店選びの一番重要なところではないでしょうか？











インフルエンザ大丈夫ですか？

大林　由衣 (2012年2月10日 11:49) 0 0 票 0 票

最近、ンフルエンザがとても流行していますね(>_<)

 

お子さんの学校でも、学級閉鎖などが多くなっているのではないでしょうか。

 私の周りでも増えています。。



つらいインフルエンザから家族を守るためには


手洗い、うがい、部屋の加湿、睡眠、栄養、マスク...

これは当たり前ですが、

 

一番大切なのは、免疫力を高める事です。

 

 

では、免疫力を高めるのに、効果的なものってなんでしょうか？


いろいろ調べてみたところ、免疫力を高めるにはビタミンDがいいらしいです！

 

ビタミンDは、主に日光浴と食事で増やすことができるそうです。

 たビタミンDが不足すると、うつ病の原因になるとも言われています。

ビタミンDは体にも心にも大切なビタミンなのです。

 

このビタミンDはキノコ類、

特に紫外線を浴びた干したキノコ類に多く含まれています。

中でも、身近な食材で非常に含有量が多いのはキクラゲだそうです。



また、魚類にも多く含まれています。

海の幸でも背骨のない海老・蟹、蛸・烏賊、貝類には含まれていません。

 

そんなこの時期に欠かせないビタミンDを

含んだ魚ときのこを効果的に取れのは

まさに、の季節にぴったりな「鍋」.ではないでしょうか？


体も温まるし、みんなでつつけば楽しく食べられます☆

 

奥様、今夜は栄養たっぷりの鍋にして、家族の健康を守りましょう♪(*^_^*)/



おいしそ?♪

自分にとって仕事とは何か

大林　由衣 (2012年2月 1日 15:26) 0 0 票 0 票
昨日は、企業合同のお食事会に参加させていただいて、

株式会社ウイッシュアップさんと
株式会社エイムさんと

「世の中に役立つ経営とは」何か

について、語りあいました(*^_^*)

私はまだまだ「経営」という観点まで考えが
及んでいないですが


私にとって「仕事」とは何かと考えたとき

「お客様の夢を実現すること」だなと
あらためて思いました。


「今のマンションではなく、
子どもがのびのび遊べて、親子がコミュニケーションを取れる
そんな一戸建てが建てたい」

というお客様の想いを、

どうやって実現するか

どうにかして必ず実現したい。


そういう想いで

決して「家を売る」のではなく

「一人ひとりの夢の実現をしていく、その手段がたまたま家なだけ」
と改めて思いました。


お客様がよくよく考えて
「家はいらない、マンションでいい」となったら
それでいいんです。

何が一番大切なのか。

それが分かって、叶えられたら
そのお手伝いをすることが私たちの仕事です。


それを改めて、語り合い、聞いていただいた
昨日は、とても有意義な時間でした。

ありがとうございました(*^_^*)

他のみなさんの 想い もとっても素晴らしく、

「自分が一人の営業マンを研修して、すくったら、
その部下も救えるし、その先のたくさんのお客様を幸せにできる」
「1万人の幸せな営業マンを作り、100万人の幸せなお客様をつくる」

など、こんな想いで全員が仕事していけば
日本は幸せになると思いました☆

どんな時でも、
「この人は何のために仕事しているんだろう」と
思ってみてみてください。

そしてこんないい想いの人を見つけて
その人に任せてください(*^_^*)
