昔の知恵に学ぶ

大林　由衣 (2012年6月 5日 15:55) 0 0 票 0 票
前回のブログに書きました
二世帯住宅の勉強会ですが

日程を変更して

6/23（土）になりそうです。

確定しましたら、御案内します(*^。^*)

二世帯住宅だけでなく、
どんな家にも必要な考え方をお話しますので

どなたもぜひおこしください（＾ｖ＾）
これを知らずに家を建てると損してしまうと
思います。。。！

家づくりは一生のことなので
しっかり知識を入れて、最高の我が家を
つくってくださいね（＾ｖ＾）/





私は先日、伊賀の忍者村に行ってきました。

そこには「忍者屋敷」の復元があって、

いろんな室内の仕掛けを
ガイドの方が教えてくれます☆


その仕掛けがとても面白く、

今の家づくりにも活かせるのではないか
昔の知恵ってすごい！！

と感動しました。


例えば、大事な書類を隠す床下収納や
刀を隠す、床板や
暗さを利用した、中からは見えるけど
外から見えない見張りスペースや、
一見棚に見える隠し階段、
壁のどんでん返しで各部屋に繋がる隠し通路、
簡単に素早く開く木の鍵など

これをそのまま活かしても「スパイの家」になりそうですが(^_^;)

最新の機械や設備に頼らなくても
こんなにすごいスパイ（忍者）屋敷が作れるんだなと
勉強になることばかりでした。

私は忍者はなんだか迷信っぽく、本当に実在した
イメージが今までなかったのですが

本当にいたんだなと思い、すごく興味がわきました。

そしてまだまだ知らない事がたくさんあるなと感じました。
知らない事を知れたいい機会でした☆

小さなお子さんは忍者の衣装を着て走り回っていて
とてもかわいかったです(*^。^*)

ぜひご家族で行ってみてください♪

近くに有名なアスレチックもあって、ご家族にお勧めです☆
