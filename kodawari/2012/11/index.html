NEWブランド　ギャラリーOPEN！！

大林　由衣 (2012年11月23日 12:30) 0 0 票 0 票

新住宅ブランドがスタートします。


その名もcasa villa「カーサヴィラ」。


Casaとは「家」villaとは「邸宅」という意味で、


デザイン・構造・住み心地・最新の設備・工法にこだわった、

まさに「毎日がリゾートのような家づくり」を展開していきます。



　弊社の母体である進和建設工業は、

主に鉄筋コンクリート造のマンションを、地元で45年間建設して参りました。


そのマンションで培った鉄筋コンクリート造の技術を、

住宅にも惜しみ無く活かしていきます。

　また、一流のデザイナーが、建物だけでなく、

家具や壁紙までトータルに監修し、どこにもない家づくりをお手伝いすると共に


耐震性・耐火性・耐久性に優れた鉄筋コンクリート造により、

外断熱工法や免震・地下室・屋上ガーデンなど、

今まで諦めていた理想の暮らしをご提供します。


　　そしてこの度、本社1階にギャラリーが完成致しました。


インテリアにこだわった、まるでリゾートホテルのような室内で、

ゆったりとくつろぎながら、あなただけの住まいの夢をカタチにしていきませんか？

　ぜひお気軽にお立ち寄りください。
