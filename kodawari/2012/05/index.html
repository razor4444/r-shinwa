2012年5月アーカイブ

今注目の二世帯住宅♪

大林　由衣 (2012年5月29日 11:47) 0 0 票 0 票
近年、親（祖父母）と離れて暮らす　
核家族化がどんどん進んできましたが、

大地震があって、家族の繋がりが最重要視されてきたことや
資金面の不安、土地の少なさなどから

最近は「二世帯住宅」が注目されています。


緒に暮らすことで、老後安心だし
お孫さんの面倒も見てもらえる。

子供もおじいちゃんおばあちゃんと
触れ合うことは、情緒やしつけに関しても
とてもいいことです。

おじいちゃんおばあちゃんと育つと
優しく、細かい気遣いの出来る子が育つとも聞きます(^-^)

また、祖父母も、小さなお孫さんと
触れ合うことで元気に過ごしてくれたりと、

二世帯住宅はいいことだらけなのです☆

今まで「二世帯って何かと窮屈そう。。」
「トラブルの元じゃぁ。。？」

などと思っていた方、意外とそんなことないんです。


今度、
「住まいのコミュニケーションを大事にする」
優しい住まいの設計士の先生に

「二世帯住宅で気をつけること、成功事例」など
とてもためになるお話を、話してもらおうと思っています☆

The 参加無料の「二世帯住宅セミナー」です(*^。^*)/

「なるほどなぁ！」「聞いて良かった。自分の家に活かそう！」
と思えるヒントがいっぱいなはず！

更に参加して頂いた方には素敵な世界に一つの
オリジナルプレゼントをご用意します！！
楽しみにしておいてください♪

日程は今のところ6/16（土）を予定しています。

また日程と詳細確定しましたら、お知らせしますね！

ぜひご参加ください！ぜったいに損はないですよ（●＾o＾●）

チャリティバザー

大林　由衣 (2012年5月23日 18:16) 0 0 票 0 票
先日5/19、
堺市八田北町でサービス付高齢者住宅の完成見学会を行いました。

屋台を出したお祭りを開催し、ご親切の方々や、近所の方が沢山来てくださいました！
ありがとうございます！

そこで、介護用品のチャリティーバザーも行って、
47200円の売り上げになりました！ありがとうございました！

こちら全額赤十字の東日本大震災に寄付させていただきます。

介護用品、格安だったと思います。買っていただいた方も、
寄付先の方もみんなが喜んでくれたらいいなと思っています。

残った介護用品もそのまま東北に寄付します。


介護用品って、どんなものがあるのか
介護に携わったことのない人はあまり知らないかもしれませんが、

とっても便利で、優しいく、アイデアのつまった商品で、
介護が必要ない方にも便利に使って頂きますよ(*^。^*)

例えば介護用の靴。

とっても柔らかく、誰がはいてもとっても楽なんです。

また、水のいらないドライシャンプーや、

タオルをエプロンにできる、型掛けクリップや

持ちやすい食器など

家族みんなで使えるなと思いました（＾ｖ＾）

ぜひ大切なひとにプレゼントしてみてはいかがですか？

介護＝少し暗いイメージがあった方
介護＝優しいイメージに変わると思います☆

省エネ住宅の補助金

大林　由衣 (2012年5月 9日 10:05) 0 0 票 0 票
今年の夏、エネルギーが不足するとのことで、
冷房などの節約が叫ばれています。

もちろん、節電は大切ですが、
我慢して熱い中を過ごすより、
もともと、エアコンを使わなくても
快適な建物が一番いいですよね。

その、快適な省エネ住宅が今特に注目されています。

省エネ住宅の1つ「スマートハウス」という
家中の使用電気をデジタル管理し、無駄なく調節する
家もだんだんと普及が始まり、

「省エネ住宅」は今、住宅の主流と化しています。

2012年今年はそんな「省エネ住宅」をもっと拡大して欲しいとのことで
政府はが省エネ住宅に様々な特約をつけています。



★今省エネ住宅を建てることお得がいっぱい★

住んでから光熱費が安く、室内も快適な省エネ住宅は
ランニングコストも安くなりますが、

初期費用もお得があります。

まず、
1.所得税減税
　　10年間で400万円お得♪
　（年間上限40万円控除　10年間で400万円控除）

2.贈与税の非課税
　　最高1610万円の贈与税525万円がゼロに！
　　（住宅取得に両親・祖母から資金支援を受ける場合
　　　その「贈与」の税金がかからなくなります）

3.金利引き下げ
　フラット35Sエコなら-0.7%
　（例）3千万円35年ローンなら　月々1万円近くもお得！

4.太陽光補助金
　　設置費用の補助　　3万円×ワット数
　（発電した電気の買い取り　42円×ワット数）

5.エコポイント
　　15万円分
　（15万円相当の商品や商品券、追加工事費用に
　　　被災地の復興支援などに交換）
　

また、新築だけじゃなくリフォームもお得です（*^_^*）

1.エコリフォームでエコポイント30万円
2.同時に耐震改修を行う際は+15万円　　
　　合計45万円分のポイント！

昔に建てた家は、今の進化した建材に比べると
断熱材や窓ガラスの省エネ効果が大変劣っている場合が
多いです。
その断熱や窓の改修を行って、性能をアップさせることで
光熱費を抑えることができます。

エコポイント詳しくは↓
http://fukko-jutaku.eco-points.jp/user/outline/

また、弊社（親会社　進和建設工業）では
耐震診断、耐震リフォーム得意です♪
http://www.taisin-reform.com/


住宅取得は、もったいないので
減税の多い時期を逃さないようにしてくださいね(*^_^*)/
