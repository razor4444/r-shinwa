大林　由衣: 2010年10月アーカイブ

建ててからの実際の感想♪

大林　由衣 (2010年10月25日 11:57) 0 0 票 0 票

先日、エコ住宅を完成されました施主様に、

竣工式をかねた、新築での食事会に呼んでいただきました♪

 

外構も全て完成し、家具も全て設置してからは初めて拝見させていただいたのですが、

とてもきれいで、感動しました。

 

外構はガレージや植栽が出来上がって、家にボリュームと迫力がプラスされ、

さらに立派に見えました。

内部はこだわって選ばれた無垢のフローリングがとてもきれいな色で、

悩んで選ばれた家具の色ともぴったりで、とても素敵でした☆

 

今まで、普通のフローリングに見慣れていましたが、、無垢材にするだけでこんなに

雰囲気が変わるということがすごく分かりました。

 

階段はスケルトンで、奥の壁に小窓がつき、玄関や、地下が見えるようになっているのが、

すごくおしゃれで、広がりも感じ、とても素敵な仕上がりになっていました。

 

一番素敵なのは、やはりリビングの吹き抜けで、

奥様もすごく気に入ってくださっていて

「空に鳥が飛んでいくところや、月まで眺めることができて、部屋もとても明るくて快適です。」

と言ってくださいました。

地下のアトリエも、地下という感じが全くせず、炭と外断熱の効果で、とてもさわやかな空気で、

気持ちよく、快適でした。

 

旦那様も

「新築独特の嫌な臭いが一切せず、初めて入った瞬間から落ち着けました。

今まで、道路を車が通るだけで揺れていたのも、まったく揺れなくなり、

外から聞こえていた音も静かになりました。窓の位置が変わり、景色もすごく良くなりました。

要望も全てかなえていただきありがとう御座いました」

 

と言ってくださり、

私は大した手伝いは何もできていないのに、涙が出そうになりました。

 

奥様も

「すごく居心地がよくて、最近はずっと家にいるんです。」

って、すごく嬉しい言葉ですね☆

 

お客様が喜んでくれると本当に良かったって思います。

喜んでもらうだけでなく、そこに住まわれる方の幸せと、家族の絆をも作れて、

そのまま一生残っていく...

 

本当にいい経験をさせてもらっていると思います。

本当にありがとう御座いました。

 

 

 

光熱費ゼロの家の証明！

大林　由衣 (2010年10月21日 09:11) 0 0 票 0 票

光熱費ゼロと言われても、

シミュレーション上のことでしょう？

家庭の暮らし方によって変わるでしょう？

本当にゼロになるの？

 

と思うと思います。

 

そこで、今回「光熱費ゼロの家casa sole（カーサソーレ）」を

実際に建てて、住まわれている方の月々の光熱費を公開しました！

 

すると、確かに立地条件（日当たり）や電気の使い方で、違いは出るものの、

各月あわせると、光熱費ゼロどころか、売電で収入が入っています！

 

多い月では1ヶ月で1万4千円もの収入が入っています！

 

詳しくはこちら↓

http://shinwahome.casasole.jp/energy/energy02

 

何もしなくても、光熱費がただになり、1万4千円もの収入が入ってくるなんて、、、

臨時収入、お小遣いになり、とっても嬉しいですね♪

 

この売電収入用の口座をつくり、貯金するのもいいし、

たっぷりエコできたつきは、家族でちょっと豪華に外食するのもいいですね♪

 

地球環境に貢献しながら、お財布も潤うまさに一石二鳥の家です(#^.^#) 

 

黒だけじゃないよ！

9.10.11日の三連休も太陽光イベントしてきました☆



三連休はダイエーグルメシティ中もず店で
たくさんのお客様が寄ってくださいまして、
本当にありがとう御座いました！


普段から太陽光が気になっていながら、費用面でためらっていた方や、
何年で費用が償却できるのか知りたい方などが自ら聞きに来てくださったりして、
いろいろご説明すると、役に立てたような気がして、とても嬉しかったです。

最近は太陽光発電、気になっている方が増えていますね♪

 

 

 

費用は大体200万円くらいです。

これを月々2万円で10年ローンにすると

 

月々2万円の電気・ガス代を払っていた家庭なら、

その光熱費の2万円が太陽光発電によって0円になり、

その浮いたお金をローンに払う感じです。

 

つまり、出費は今までと変わらず、太陽光がつけられるのです。

さらに、10年でローンが払い終われば、あとは一生光熱費0円！！

しかも余った電気を売って収入まで入ってきます！

 

200万というと、「高い！！」ってイメージを持つかも知れませんが、実際は

つけたほうが、ものすごく安くなるのです。

 

だって、10年後0円が20年続くのと、30年間ずっと光熱費を払うのと比べると

480万円もの差が生まれます！！

 

480万も、光熱費に消えていくなんて、考えただけであほらしいですよね(^_^;)

 

太陽光は今年が一番補助金がたくさん出ます。

さらに弊社は今年限定で、ほぼ原価の利益なしの激安価格で提供しています！！！←ホントです(^^;)

 

また、新築をお考えの方は、はじめから太陽光とセットの家が一番効率がいいですよ♪

（太陽光とセットで、その発電量を余すとこなく活用できる家→http://shinwahome.casasole.jp/）

 

 



来週と再来週も、ダイエーグルメシティ中もず店（1ＦUFJ銀行横入り口入ってすぐ）
でしていますので、お立ち寄りの際にはぜひ♪

お待ちしています♪

宇宙ステーション？

 

 

 

実はこれ、アブダビの海にそびえる空中農園だそうです。

 


 

 

「総人口80万人以上の砂漠都市・アブダビでは、真水を得るために5つの大型塩分除去工場を利用し、新鮮な果物や野菜は輸入に依存している。

 

イタリアの建築設計事務所「STUDIO　MOBILE」はこのほど、これら全ての問題を解決する、「垂直農場」プロジェクトを提案した。

　このプロジェクトは、マユのような形をした5つの温室が一本の柱につながっているというもの。

それぞれの「マユ」の中では、海水が水蒸気となり、温室の冷却・加湿作用を果たしている。このほか、海水は蒸留作用で真水に変えられる。」
