今更聞けない「エコ住宅」についておさらい♪

大林　由衣 (2010年9月15日 12:38) 0 0 票 0 票

最近ＣＭでもよく目にするエコ住宅（省エネ住宅）ですが、



今更聞けないけど......


いったいどこが、

何が「エコ」なの！？

 

 



ということで、今回は一から「エコ住宅」についておさらいしたいと思います。

 

 

 


「エコ住宅」について思いつくこと、それは...


値段は？　　　「エコ」って付くだけで高くしてるんじゃないの？
住み心地は？　節約＝我慢じゃないの？
仕様は？　　　安っぽいんじゃないの？
デザインは？　ダサいんじゃないの？(>_<)

てこと。。




第一に「本当のエコ住宅」とは具体的に言うと、


・断熱材を分厚くし、夏の暑さ、冬の寒さを室内に伝えないようにし、冷暖房費を削減した住宅。

・いちばん内外の熱を伝える窓ガラスを二重にし、熱を逃がさないようにした住宅。

・家庭で一番エネルギーを使う給湯器を最新の省エネ機器に変えた住宅。

・ＬＥＤや省エネ性能エアコンなど、電気代を削減できる機器を使った住宅。

・メンテナンス、建て替えで資材やお金を使わないようにした長期住宅。

・太陽光パネルなどで自然エネルギーを利用し、ＣＯ2を発生させることなく自家発電する住宅。


などになります。




そしてまず、値段。

これは「エコ」という響きに無駄に高くしているのではなくて、

エコの住宅をつくるために、上記の仕様にするにはどうしてもお金が掛かります。

しかし今なら、ありがたい「住宅エコポイント」という制度があるので、

上記の仕様にするとポイントが貰え、それを工事費に当てることができます。



そして住み心地は

省エネ＝節約＝我慢　ではなく、冷暖房費を削減するために

断熱効果を高くしますので、自然と快適に、冷暖房の機械設備に頼ることなく、

健康に暮らせます。まさに、省エネと健康の一石二鳥なのです。



次に仕様は

エコばかりにこだわり、安っぽいものではなく、最新の高機能なものです。
どの設備メーカーも、最新のものはすべてといっていいほど、エコ仕様にして開発しています。
そのため、エコ住宅は最新の設備を使った高機能住宅になるのです。



デザインは

エコにこだわるならデザインは捨てなければいけない...ではなく、

デザインはもちろん自由♪自分好みにかっこよく、大胆にデザインしてください。

大きな部屋や、大きな吹き抜け。今まで、冷暖房費がかさむからとあきらめていた

デザインも断熱性能を挙げることで、苦なくデザインすることができます。

逆にデザインの幅が広がるってことですね♪



ぜひぜひ、これを踏まえ、エコ住宅に注目してみてはいかがでしょうか？(#^.^#)

エコポイント対応の家見学会♪

大林　由衣 (2010年9月 7日 15:25) 0 0 票 0 票

エコポイント対応の家がまもなく完成します♪

エコポイント対応とは？

 

エコポイント30万円相当がもらえるのはもちろんのこと、

 

そのポイントをもらうためには、家を「エコ仕様」にしなければいけません。

 

断熱材を分厚くしたり、ガラスを断熱性の高いlow-e（ロウイー）ガラスにしたり、給湯器を高効率なモノにしたり...

 

この仕様基準をクリアすることで、冷暖房費が下がる「エコな」家が完成し、結果ポイントがもらえるということです。

 

ポイントを上げてでもしてもらいたい「政府推進仕様の家」というわけですね♪

 

 

さらに、今回のおうちは、エコポイント仕様にさらに、エコな設備が盛りたくさん♪

太陽光パネルに、エネファーム、炭の断熱材や、地下室の外断熱など、

 

最新のエコ設備を余すとこなく取り入れた本当のエコ住宅となっています。

 

 

これから建てる家は間違いなくこうなっていきます。

ぜひ一度来てみて、ご覧下さい♪

 

日程は10月頭ごろを予定しておりますので、決まり次第お知らせします☆

9月、新学期☆子どもを勉強させる方法

大林　由衣 (2010年9月 1日 09:15) 0 0 票 0 票

ついに今日から9月になりました。

9月といえば秋のイメージですが、あいもかわらず今日も暑いですね(>_<)

空も入道雲がもくもくと...まだまだ真夏のままですね

 

９月といえば、新学期の始まり！ですが、

みんな夏休みの宿題は終わっているのかな？

 

 

夏休みの間中、「宿題しなさい！」と何度も言った親御さんも多いのでは...

 

最近お客様から「子どもが勉強しなくて困っている」という悩みをお聞きしました。

 

そこで、どうやったら子どもが勉強するようになるのかをいろいろと調べると

子どもがすすんで勉強する家庭と、子どもが勉強嫌いな家庭の根本的違いがわかりました。

 

今まで意識しなかったこと、普通だと思っていたことが、大きな間違いだった。

接し方、話しかける言葉の選び方一つで、子どもが進んで勉強するようになった。

 

など、

 

今までどこの家庭も同じだと思っていたら、こんなにも違っていた。

しかもこれくらいの違いなら、今からでもすぐに変えられる。

ということがわかりました。

 

 

その内容は、たとえば、

「勉強しなさい！」というのは実は逆効果。

勉強できるこの家庭では、家族でキャンプに行っている、親が一緒にゲームをしている。

 

 

など、驚きの内容です。

 

 

また、すすんで勉強する習慣はなるべく幼いころから、なるべく早めに付くようにしなければなりません。

 

そして、ただ勉強をするだけでなく、人間として豊かな人格に育ち、みんなから愛され、仕事も成功するということ、これは、本当に正しい勉強の仕方の習慣をつけさせることとつながっています。

それは、

勉強を好きにさせるために家族コミュニケーションをよくする＝勉強して頭がよくなる＝考える力が付く＝相手の気持ちを考えられるようになる

 

というループがあるからです。

 

そうやって、勉強を楽しんでし、忍耐力と想像力と向上心を身につけ、周りとのコミュニケーションの上手いお子さんになるかどうかは、親御さんにかかっています。

 

 

具体的にどうやったら「勉強しなさいといわなくても、子どもが進んで勉強するようになるのか」を知りたい方は「勉強法　希望」と気軽にメールください♪
