竜巻にも地震にも負けない家！

大林　由衣 (2013年9月 6日 19:38) 0 0 票 0 票
最近異常気象が進行し
竜巻やスコール、地震など、日々心配で
うちの家は木造だけど大丈夫かな。。。とよく考えてしまいます。


とくに、最近大きな被害が出た竜巻は珍しく
対処法もわからないので不安・・・

ということで「竜巻の時の対処法」を調べてみました！


竜巻が近づいたら！
  • 筋コンクリート造のビルなど、近くの頑丈な建物に避難しよう。
  • 近くに頑丈な建物がない場合は、近くの水路やくぼみに身を伏せて、両腕で頭と首を守ろう。
  • 車庫、物置、プレハブ（仮設建築物）などへの避難は避けよう。
  • 避難時は飛んでくる物に十分注意しよう。
  • 地下室や1階の中心部に近いところ、地下室があれば地下室に。
  • 窓の少ない部屋に移動しよう。
  • 窓やカーテンを閉め、窓際から離れよう。
  • 部屋の隅、ドア、外壁から離れよう。
  • 丈夫なテーブルや机の下に入り、両腕で頭と首を守ろう。
※ＨＯＮＤＡみんなの防犯ノートより


やはり、コンクリートの建物が安心ですね！

そして　地下室が安心！ということで

なるほど！！

では地下付きコンクリート住宅に住みたい！！！

ということで、、、、

あります！そんな理想の住宅が！

しかもデザインもすごい！世界で活躍する黒川建築士がつくった住宅です！
