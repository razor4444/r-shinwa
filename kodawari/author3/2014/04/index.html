進和ホーム: 2014年4月アーカイブ

愛を持って人を幸せにする

進和ホーム (2014年4月30日 08:16) 0 0 票 0 票
人間は誰しもが幸せになりたいと願う生き物であり、
それが人生の究極の目標であるとひしひしと感じた一日でした。
私の場合は、自分自身の目標のために
住宅という仕事を通じて世の中の人々を
一人でも多く幸せにすることが使命だと感じております。
そのためには日々の勉強とスキルアップが必須ですが、
それ以上に大切なものがあると思います。

『人生・仕事の結果＝考え方×熱意×能力』
これを進和では掲げています。
この考え方が最重要であり、愛を持った利他の精神が
仕事をする上で、また人生においても
非常に重要だと感じた貴重な一日でした！
　　　　　　　　　　　　　　　　　　　　　　　　　　佐野

人のために尽くす

進和ホーム (2014年4月24日 18:35) 0 0 票 0 票
このタイトルは進和建設工業の
大切にしている精神です。
また今日一日、外を回って話を
伺うと、自分がどれだけド真剣に
相手のことを考えられているか、
自分を犠牲にしてまでも相手を
考えた行動が取れているか
ということを自問自答しました。
やはり、まだまだ不十分な点が
見つかりました。
仕事だけでなく、人生も成功するために
自分自身の姿勢を再度見つめ直し
住宅を通じて、人を幸せにしていくために
日々精進していきます！
                                                 佐野

自利利他の精神

初めまして。進和の革命児です。

進和ホーム (2014年4月21日 18:36) 0 0 票 0 票
初めまして！四月一日付で進和建設工業に入社しました
佐野暢亮（のぶあき）と申します。
実家が造園業で桜に特化しているため
祖父が暢桜をつくってくれたのが由来です。
新入社員研修が終わり、進和ホームに配属が決まり、
住宅を中心に建築界に『革命』を起こしていきます！

まだまだ若輩者ですが頂上からの景色を眺めるため
皆様に仕事を通じて感動を与えていきます。
どうぞ宜しくお願い致します！
                                                         佐野
