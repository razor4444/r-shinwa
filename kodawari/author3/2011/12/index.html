お引き渡し☆

大林　由衣 (2011年12月 7日 14:08) 0 0 票 0 票

12/3.4に開催させていただきました

完全バリアフリー「優しい家」の見学会が無事終了しました。

 

そして本日、お施主様に引き渡しをさせていただきました。

 

見学会のご協力並びに、

大事な家を私共に最後まで任せていただきまして、

本当にありがとうございました！

 

こんなにいい家が出来て、私共も大変嬉しく思っています。

 

ご家族の想いがつまった家は、とっても温かく、

落ち着きました。

 

これからも、お困りごとがありましたら、

いつでもよんでください。

 

これからも一生のお付き合いよろしくお願いします。

 

 

 

進和ホーム　一同

好評です！

大林　由衣 (2011年12月 3日 22:46) 0 0 票 0 票

本日、T様邸住宅完成見学会の1日目を開催しました！

 

雨で、お足もとの悪い中、

お客様やご近所の方など、たくさん来ていただきました。

ありがとうございました！

 

家の中では、

設計士の先生が、この住宅にかけた想いと、

工夫点を、一人ひとりに細かく、熱く語りました。

 

段差のないバリアフリーはもちろん、

コンセントの位置、

スイッチの位置も、

車いすから押しやすく、

電球も取り変える負担を減らすため、

半永久型のLEDに。

 

目線の抜ける壁の配置や、

天井の隅を折り上げることで、

空間が広く、天井が高く見える効果などなど。

 

風通しと、光と緑の取り入れ方にも、

こだわって、初めて入った家なのに、

もの凄く落ち着くという。とってもすごしやすい、

「優しい家」です。

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

見学いただいた方からも、

「とても手の込んだこだわりのお家みたいで、

勉強になりました」

と言っていただきました。

 

細かなこだわりが、

こんなに過ごしやすい本物の「住まい」をつくります。

 

ぜひ参考にしていただきたいなと感じました。

明日も開催しています！

 

ぜひお時間ございましたら、

のぞきにきてください♪

 

きっとご自分の家づくりに、

いいアイデアが見つかりますよ！

 

緊急告知♪見学会のお知らせです☆

大林　由衣 (2011年12月 1日 13:29) 0 0 票 0 票

あっというまに12月に入り、すっかり寒くなってきましたが、

みなさんいかがお過ごしでしょうか？

 

 

このたび、堺市堺区に

注文住宅が

無事完成しました！

 

そこで、見学会を

開催しますので

ぜひお越しください♪

 

12/3（土）4（日）

10：00から16：00

場所：大阪府堺市堺区

今池町2-3-20隣

 

 

このおうちの特徴は！！？？？

 

「老後もゆっくりと暮らせる家」ということで

設計デザインで、「住み心地」に、とことんこだわりました！

 

西から東へ風の通り道を作って風通しを良くして快適に、

 

食卓からは庭の木々花々が見れるようにしました。

 

そして、老後に一番気になるのは

「バリアフリー」ではないでしょうか？(*^_^*)

 

この家は完全バリアフリーはもちろん、

そのほか、随所に安全性こだわった、やさしい家です♪

どんなところが工夫点か、ぜひ見て見つけてください♪

 

そして、一番のこだわりは、

実は目には見えないところ...

断熱性です！

家の中の温度変化を最小限に抑えて

快適な暮らしをするために、

断熱効果の高い、

○○という断熱材を使用しています。

 

これはどんな断熱材かというと.........

 

続きは、来ていただいた方に詳しくご説明します(*^^)v

 

また、ぜひこの寒くなった季節に、実際に室内に入って、

温度の違いを確かめて

みてください♪

 

 

当日は、この家の担当の設計士もいますので、

ご自分の家の悩みなど、なんでも気軽に質問・相談してください♪

 

 

 

 

見学をご希望の方は下記までメールか電話で予約ください♪
