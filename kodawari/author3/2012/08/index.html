スマートハウス！とは！？

大林　由衣 (2012年8月31日 11:44) 0 0 票 0 票
今話題のスマートハウス！！
みなさん、どんな家なのか、詳しくご存知ですが？

最近よく聞く「スマート」とは「賢い」という意味です。

では「賢い家」とは？


「快適に節電しながら暮らせる次世代のエネルギーハウス」です。

例えば
昼間太陽光発電で発電し、蓄電池や電気自動車に蓄電し、
その電気を夜間も使えるようにします。

家電も節電、節水に優れたものにし

家事態の省エネ性、（断熱材やメンテナンス費用の少ないもの）
もアップさせた構造に。

また、コンピューターが家全体の使用電力を監視し
各部屋の使用電力が一目で見れたり、
普段よりも使いすぎの部屋は警告が出たり、
省エネ目標設定などもできます。

また、その電力の管理をパソコンやスマートフォンで
離れたところから見たり、
電気や家電のスイッチを遠隔操作し
消し忘れや、使いすぎを防げます。

また、スマートハウスの集合区画では
発電した電力を分け合い
災害時でも助け合い、安心な生活ができる仕組みも作られています。


まさに、大地震以来の電力問題の解決策になる
次世代の家ですが

こういった最新設備がそろうと、どうしても
値段が高くなるというデメリットがあります。

そこで、標準的な価格でできる「スマートハウスパック」を
ご用意しました。

無駄を省き、低価格で最新の設備をそろえ、
快適に暮らせる字次世代スマートハウス
「ナレッジ」を一度ご覧ください(*^。^*)

今お考えの住宅にそのまま機能と設備を載せることも可能です♪


パンフレットを無料で差し上げます♪

家の中身を見に来ませんか！？

構造見学会のお知らせ♪
堺市西区で注文住宅が施工中です。


CUBE型のデザイン注文住宅ですが、

正面から見ると窓がありません！


なぜ窓をつけなかったのか？

それは、この外壁を「キャンパス」にしようと考えたからです。

家の正面に木が生え、夜は地面から照らされた

ライトが木にあたり、その影が壁に映し出される・・・


そんな素敵な住宅です。



先日無事上棟しました。 
みなさん工事にご協力ありがとうございます。

今なら、完成してからは見ることが出来ない
壁の内部や、構造体が見られる
構造見学会を開催中です。

施工中の現場ですので
安全のため完全予約制ですので

ご興味のある方ごお問い合わせください。
