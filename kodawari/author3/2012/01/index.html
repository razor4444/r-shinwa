大林　由衣: 2012年1月アーカイブ

子育てを成功させるために

大林　由衣 (2012年1月28日 16:29) 0 0 票 0 票
こんにちは！

進和ホーム　大林です(*^_^*)

最近「幼児教育」について調べています（＾ｖ＾）


子供を天才に育てるのは、
0?3歳の教育が大切だそうです。


「人間の能力は、年齢とともに増えて行くのではなく、
生まれたときが一番容量が大きく、段々と衰退していく
から幼児のうちが一番たいせつ」

ということで

0?3歳の教育とは、
右脳を育てる教育、
親子が関わりしみながら共に学び育っていく教育

また、おなかの中に居る時からできる胎教。などなど


気になる情報がたくさんあります。


また、「うちのこはもう大きくなっているし、幼児教育は間に合わない」
という方には

「子供を勉強させる方法」をまとめた小冊子があります。



勉強嫌いで、ゲームばっかりしているこをどうにかしたい！

勉強好きと勉強嫌いな子供の親の違いってなんでしょう？


それを知ると、

「なるほど！今まで逆効果なことばかりしていた！」

と思うことがたくさんあります(>_<)


そして、もっと早く知っておけばよかった。
きっとそう思います。(^_^)


子育てを成功させ、子供に幸せな人生を歩んで欲しい。

全ての親はそう願っていると思います。


私も、会に役立つ立派な人が一人でも多く育ちますように。
社会が幸せな家庭でいっぱいになりますように。
と願っています。

なので、この「子供を勉強させる方法」の小冊子を作成し、
興味のある方にプレゼントしています。



気になる方は、お気軽に
「小冊子希望」とメールください♪(*^_^*)

車を楽しむ家っていいな！

大林　由衣 (2012年1月24日 16:55) 0 0 票 0 票

お久しぶりです！

 

 

明けましておめでとうございます。



...って、あっと言う間にもう1月も終わりそうですね！

 

ずっと更新出来ていませんでした。

 

すみません(;_:)

 

最近寒さがますます厳しくなってきましたが、みなさん、体調など気をつけてくださいね。

 

 

 

先日私は「大阪モーターショー」へ行ってきました。

すごい人が多くて、車一つみるのにも一苦労でしたが...

なんとか頑張って写真を撮ってきました。

 

私はペーパードライバーなんで、車の性能については全然わかりませんが

車って、芸術的なので好きです。

限られた形の中で、美しいデザインを限りなく作り出せることが凄いなぁと思います。





 

 

あんな美しいデザインの車が、毎日眺めて暮らせれたらなぁ

車もインテリアの一部にできれば。。

 



そんな夢をかなえるのが、ガレージ付きの家ですよね。

とくにお勧めのcasa cubeのガレージ付きタイプは、

家がシンプルなので車が映えるし、

家の中から車が見えるし、車庫にもすぐに行き来できます。

 

趣味を存分に楽しみながら、毎日安らげる暮らしをぜひ(*^_^*)
