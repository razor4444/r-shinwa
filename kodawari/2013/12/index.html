堺市東区のK様邸、一階二階の建て方が完了しました＾＾

現場には火打ちや梁がガッチリ組まれた、綺麗なキューブ 型の建物

になっています。

 

前回、お施主さんが来られたときは、基礎しか立ち上がっておらず、

家の間取りのイメージがつきませんでしたが、今回は立体的なイメ

ージもわきますし、家のかたちも具体的になり、ご夫婦が描く住まい

が、少しづつ出来上がっていきます。

 

次の工程が楽しみですね＾＾

 

掃除ラインと、効率を考え手順よく

大重 (2013年12月24日 22:44) 0 0 票 0 票

お掃除ラインを考えて行うと、時間も労力も無駄にせず、効率的な大掃除ができます。

 

まず、家の間取り図を頭に思い浮かべながらイメージしてみましょう。

 

 

そして、どこの部屋から始め、どう回るか、お掃除ラインを決めます。

 

次に、手順ですが、ホコリは下へ落ちますので、天井→照明→カーテンのホコリ、家具、テーブルの順番で上から下へホコリをはらいます。

 

家具を拭くときは汚れた家具からではなく、部屋を時計回りに歩いて拭くなど方向を決めると動きに無駄が出ないので順番としてもいいでしょう。


そして、掃除機は部屋の奥から手前へかけるのが基本。プラグを入れ替える手間も考えて，あらかじめ延長コードを用意しておくと、プラグを取り替える手間も省けスムーズに移動できます。もちろん、どの順番で部屋を回るとスピーディか、確認することをお忘れなく。

吹き抜けのある家?寒さ対策

大重 (2013年12月24日 22:32) 0 0 票 0 票

リビングなどに吹き抜けがあると、その開放感は素晴らしく、なかなか他の間取りでは得難いものがあります。

吹き抜けは、施主さんの間取りに関するアンケートの中でも常に上位の人気があります。

 

しかし、寒さ対策は十分にしておかないと、実際に住んでみて後悔することになってしまいます。

日本の気温の変化を想定し、気密性を高めるとともに、断熱材やサッシ、ガラス、それに暖房器具の選定も慎重に行った上で、間取りにも気を配らなければなりません。

 

高気密高断熱にこだわり設計していても、住み始めてからは、夏・冬のシーズンのほとんどの時間で冷暖房を使用している、という声をよく聞きます。

 

その対策として、まず設計時に設けておきたいのがシーリングファン床暖房なら必要ないかもしれませんが、エアコン暖房ならシーリングファンは必須でしょう。

 

夏場は風を下向きに送り、冬場は風を天井に向けて送ります。　こうすることで、室内の空気を撹拌し、室温を均一にしてくれます。

 

シーリングファンが無い場合は、フロアタイプの送風機等をうまく活用しましょう。

冬の冷え込みに＾＾

大重 (2013年12月24日 22:25) 0 0 票 0 票

本格的な冬が近づいてまいりました。明日はこの堺市でも最低気温が3度になり、みぞれが降るとの予報です。この季節になると、暖房が効かないですとか、窓際がひんやりすると、いったお声をよく聞きます。

 

どれだけ壁で断熱していても、熱の最大の移動部は窓 です。

 

 窓を閉め切った状態で一枚ガラスの場合、冬に暖房した熱は逃げ、. 逆に夏には 外の熱が入ってきてしまいます。 気が付かないうちに、エネルギーのロスは想像以上 に多くなり、部屋うちを冷やしているのです。

 

最近では主流になりましたがガラスが2枚のペアガラスに変えると、断熱効果も大幅に増し、付け加えアルミの窓枠を樹脂サッシに交換することで、さらに効果を発揮します。

 

窓際の結露がひどい、窓にちかよりたくない。なんて思ってらっしゃる方は一度検討してみたはどうでしょうか？

 

 

 

 

 

 

 

２０１４年度の税制改正案、所得控除縮小も。

大重 (2013年12月13日 19:26) 0 0 票 0 票
先日、２０１４年度税制改正大綱で、来年４月の消費税引き上げをにらみ、
会社員の給与所得控除の縮小や、地方法人税の見直しなどが行われま
した。

年収１０００万円を超える会社員の所得税や住民税の負担が段階的に増
え、２０１６年からは年収１２００万円超え、１７年以降は年収１０００万円
超の給与所得控除が縮小されるとの事。

所得控除が減って、１４年には消費税が８％にがあがり、２０１５年には
１０％になろうとしています。

住まいに関していえば、９月に駆け込み需要があり、これから年収や家
族構成によっても住宅購入時期を考えないといけないようになりました。

住宅購入金額に対し増税はローンの支払い負担になりますし、毎月の
事ですからさまざまな部分で支出は増える一方になるでしょう。

私たちは住まいについて、少しでもその負担を軽減できるような提案を
心がけています。

本当に厳しい時代入ってきましたが、これも日本の財政建て直しのため
と考え、企業努力と提案力に磨きをかけ、少しでもお客さまに喜んでいた
だけるような提案の内容を考え、これからも頑張っていきます＾＾
