大重: 2014年1月アーカイブ

住まいでできるインフルエンザ対策

大重 (2014年1月20日 09:08) 0 0 票 0 票

普段できることで簡単予防

 

予防の方法として積極的に行ってほしいことは、

その日着ない上着・コートやスーツなどは日光

に当てて干すことです。


インフルエンザウィルスは日光に弱いといわれ

ます。冬の直射日光で数十分照射すれば死滅す

るという説もあります。また晴れの日には感染

者に近づいても、２メートル離れていれば感染

しない、という話も聞きます。
インフルエンザシーズン、人ごみで着た上着は、

通常の洗濯物を屋外に干す際にでも、一緒に日

光にさらすとより安心です。



空気の入れ替えで簡単予防


冬の室内の空気はたいへん汚れやすくなってい

ます。

暖房器具から出る汚れた空気のほか、着ている

衣類ら出るホコリも空気を汚す一因です。


汚れた空気は気管や粘膜を傷めるため、抵抗力

を落してインフルエンザに感染しやすくなります。


午前中早い時間の澄んだ空気は適度に湿気を含
んでる為、冬場の室内換気にとても適しています。
天や時間を見て、部屋の空気を入れ替えるよう
にししょう。
　


住まいの中から始められるノロウィルス対策

大重 (2014年1月18日 21:22) 0 0 票 0 票

　先日浜松で、給食の食パンが原因で、児童がノロウイルスに感染し、食

中毒になったというニュースがありました。

インフルエンザ同様、冬の風物詩ともなっている「ノロウィルス」による

胃腸炎。この病気に、住まいでできる対策をお伝えします。

インフルエンザと並ぶノウィルス。今年もすでに流行の兆しです。

いつ感染するとも知れないウィルスは、抵抗力の弱い子どもや高

齢の方にとっては脅威でしょう。


主に子ども絡みで感染するノロウィルスと、家の中で対峙するコ

ツをお伝えする前に、症状と惨状を先にお伝えします。。

 

ノロウィルスのもたらす症状と惨状

ノロウィルスの潜伏期間は、およそ36時間（１?２日）です。抵抗

力のある大人などでは、「漠然とした不快感」のみといった症状で済

むこともありますが、多くは嘔気を感じたり、嘔吐・下痢・腹痛をと

もなった突発的な症状に見舞われるようです。

先日は弊社の社員も、子どもからウイルスをもらい、２日お休みしま

した。

乳幼児に罹りやすく、流行りやすいのは、子どもは大人のように嘔気

のコントロールが効かず、場所を問わずに嘔吐するため。

この嘔吐物や痢などの始末の際に親も感染。その後家族全員ダウンと

いうパターンはどこのご家庭でも定番です。

とはいえ、よほどひどくなければ２,３日で回復します。

インフルエンザの季節です。

大重 (2014年1月17日 08:57) 0 0 票 0 票

今年のインフルエンザはA型が流行するというお話をよく聞きますが、

インフルエンザに限らずに冬の健康管理に有効な、家で始められる環

境ポイントについてお伝えします。どこのご家庭でもできる方法なので、

ぜひ早めに取り入れてみて下さい。


?玄関に上着・コートかけを準備

屋外、特に人が集まる駅や電車、バスなどを利用している人の上着に
は、インフルエンザのみならず、風邪をひいている人の咳やくしゃみな
どの飛まつが付着しています。

１回の咳によって飛び散る鼻汁や気道分泌物（唾や痰など）の小粒子
は10万個、くしゃみでは200万個と言われ、聞いただけでビックリする
数値が報告されていることもあり、このウイルスが付いた上着やコート
を室内などに持ち込むことは避けたほうがよいでしょう。
　
屋外から帰ってきたら、玄関前か玄関先で上着を脱ぎ、かけておくコ
ーナーを準備しましょう。コート掛を置くことのできない玄関の場合、ド
ア上部にかけることのできる簡易フックなどがとても便利です。

とにかく室内に持ち込まないような工夫をし、うがい、手洗いをしてから
室内に入れば、効果はより高まります。
