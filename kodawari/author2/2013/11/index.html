大重: 2013年11月アーカイブ

11月4日は地鎮祭でした。

大重 (2013年11月 6日 21:13) 0 0 票 0 票

新築物件の着工です。

今回は、堺市東区で建築させて頂きます。

前日からの雨で、この日も晴れるかどうか心配でしたが、なんと！

9時からの地鎮祭を前に、晴れだしました。

テントの中がぬかるんでいたので、布で雨を吸い込んだり、人工芝を持ってきたりと、応急処置も大変でしたが、業者さんの協力もあり、無事に準備が整いました。

 

 

その後、施主のＫ様ご夫妻、ご夫妻の両お父様、両お母様

ともにご参加いただき、とどこおりなく終了いたしました。

 

地鎮祭は土地神様を鎮め、土地を利用することの許しを得る。

 

安全祈願祭と呼ばれることもありますが、ご夫妻初めてのこと

なので、ひとつひとつの儀式に戸惑いながらも、楽しんでおら

れるようでした。

 

その時の写真は今お見せできませんが、施主様のご了承を

頂いた際には、ブログアップさせていただきます＾＾

K様邸の防水工事が始まりました。

大重 (2013年11月 1日 19:45) 0 0 票 0 票

今日は久しぶりの快晴で、気温も高く最高の１日でした。

K様邸では、防水工事が始まり、外壁の下地の工事もはじまり、

これからどんな雰囲気になっていくのか。工事の進み具合が楽しみです＾＾　　

 

本格的な木造建築なので、骨組みのうちに見れるだけ見ておきたいものです。

これだけの規模で建てると、普段見れないような、大工さんの隠し技を発見で

きるかもしれません＾０＾/

職人さんの邪魔にならないよう、しっかり勉強させていただきます。
