資金計画セミナーのご感想

大林　由衣 (2011年1月27日 16:54) 0 0 票 0 票

昨年12月に

FPの先生による資金計画セミナーを開催しました(#^.^#)

 

その後、個別に、「ライフプラン」と呼ばれる

その人のためだけの資金計画グラフを

FPの先生と一緒に作ることができるのですが

 

前回セミナーに来ていただいて

最近このライフプランをお作りになったお客様から

 

「ライフプラン作成、とても有意義でした。先生も信頼できる方だし。

今後も宜しくお願いします。」

 

と、嬉しい御感想を頂きました(^_^)

 

ありがとう御座います！

私は何もしていないのに、なんだか私もとっても嬉しかったです(#^.^#)/

 

この資金計画の機会が、お客様のマイホームの夢へ

近づくお手伝いとして役立て、とても嬉しいです☆

 

また、資金計画セミナーを行いたいと思いますので

前回来られなかった方もぜひお越しくださいね(^_^)

便利な近未来★

大林　由衣 (2011年1月14日 09:01) 0 0 票 0 票

おはようございます(#^.^#)

 

最近はやっとお正月気分が抜け、

寒さが日に日に増していく感じがしますね。

 

昨日ちょっとだけショックなことがありました＝3

 

 

夜家に帰ったら、家がなんだか暖かい...

 

 

暖房消し忘れていました！

 

ショック(>_<)

 

 

たまに猫がリモコンを踏んで

かってについてるときもあるんですが、

今回は朝消した記憶がないので、自責です...

 

 

こんなとき、

「猫がまたリモコン踏んで消してくれたらいいのにー」

とか思ったり。

 

まぁ、それは無理ですが、

 

 

 

これから、住宅には便利なシステムが導入されていくそうです。

 

 

まず、電化製品のリモコンの一元化です。

今まで、メーカーも規格もバラバラだった電化製品を

携帯のスマートフォンを使って、リモコン操作できるようにします。

 

 

すると、外出先からも操作できるようになり、

消し忘れはもちろん、家に着く前に、

エアコンのスイッチを入れたり、

お風呂を沸かしたりできるそうです☆

 

 

そして、家中の電化製品の電気消費量を

テレビ画面に映し出したり、

その中で、無駄を見つけて省エネしたり

できるようになるそうです(^_^)

 

 

いいことですね☆

 

 

未来の技術はどんどん進んでいます★

皆様により良い暮らしをお届けしたいと思います(#^.^#)

 

 

 

 

 

明けましておめでとうございます

大林　由衣 (2011年1月 6日 09:43) 0 0 票 0 票

新年明けましておめでとう御座います☆

 

今年も宜しくお願いいたします。

 

 

今日から仕事始めです(^_^)/

今年は少しゆったりめのお休みでした。

 

今日は全社員で百舌鳥八幡さんに

初詣に行きました。

お払い、ご祈祷をしていただき、お神酒も

頂いて、みんなで写真も撮りました。

 

今年一年も無事に過ごせ、実りのある年に

なりますようにと願いを込めて...(#^.^#)

 

そして、進和に関る全ての人が幸せになれますように☆

 

今日は営業の皆さんは挨拶廻りをしまして、

夕方からは毎年恒例の新年会です。

 

ついこの間、忘年会をやっていただいて、

新年会まで、ありがたいです☆

 

 

今年の個人的抱負は飛躍の年にすること。

忘れずに頑張って行きたいと思います。

 

皆様本年もどうぞ宜しくお願い致します。
