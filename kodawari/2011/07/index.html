ドラえもんの道具が実現！？

大林　由衣 (2011年7月21日 21:05) 0 0 票 0 票
今、大阪梅田のセンタービルで

ドラえもんの科学未来展が開かれています。

ドラえもんのまんがで出てきた道具を
科学で今、もしくは未来に実現したら
こうなる！

というわくわくするような内容です。

そういう発明や化学とドラえもんに目がない
母と私で、子供たちの中に混ざって
見に行ってきました。

意外にいろいろな道具がすでに実現している！？
ことにびっくり（＾▼＾）

一番おどろいたのが

「消えるマント」です！！

光を曲げて
そのマントをはおった人は見えずに、
その後ろが見える！という

驚きの発明です！

順番に並んで記念撮影していました。
本当に消えていました！

これは007の消えるボンドカーとかも
実現するな！と思いました。

あとは竹コプタ―に似た一人ヘリコプターや
タイムマシンの作り方などなど


ここに来た子供たちの中で、
刺激を受けて、
将来こんな人類や地球や宇宙のための
莫大な研究をする人に育ってくれたらいいなぁ。

と感じました。


大山崎美術館

大林　由衣 (2011年7月19日 09:18) 0 0 票 0 票
先日は
京都府乙訓郡大山崎町にある
大山崎美術館に行ってきました。


近くに、サントリー美術館があり、
こちらは「アサヒビール大山崎美術館」。

山崎なのかアサヒなのか、なんだか
混乱しそうな名前ですが...

JR山崎駅から徒歩で少し山を登ったところにあるの
ですが、行きは上り坂なので無料送迎バスが有り、
お勧めです。


山の木の木陰は涼しいですね☆
普通のテントなどの影と違って、
はっぱが呼吸して、水蒸気をはき出しているから
涼しいらしいです。

自然の力はすごいですね☆



美術館は
モネの睡蓮をモチーフにした中庭がとっても
素敵でした。

古風な建物の隣に、安藤忠雄さんが設計した
コンクリートの建物が地下に建設され、
回廊のみ地上から見られます。

なんだか不思議な組み合わせです。


また、隣にさらに建設中だったので、なにかできるのかも。
楽しみですね。



今回の展示物は、いろいろな椅子。
ほぼ全て、実際座ることができ、
その中で一番のお気に入りを投票しました。


親子連れの方も、お子さんが張り切って
投票していました。
小さなころから、芸術に触れる機会をつくって
あげることは大切だと思います。


また、安藤忠雄さんがつくった
地下の展示場にはモネの絵が常設されてます。



こんなきれいな庭があったら癒されるし、
涼しいだろうなぁと
のんびりできた瞬間でし
た。

美術館ＨＰ
http://www.asahibeer-oyamazaki.com/

雑誌『PEN』とは？

大林　由衣 (2011年7月13日 09:07) 0 0 票 0 票

阪急コミュニケーションズから発行されている

７月号の『PEN』にCASA CUBEの広告が載っています。

 



PENとは


 術・宗教・建築・歴史・自然科学・旅・料理・家電・

ファッション・デザイン・車・・・地球上のあらゆる事象が特集のテーマとなり、

本当に「いいな！」と思えるモノを特集しています。

 


それぞれのテーマを独自の視点で深く掘り下げながらも、

わかりやすく楽しい紙面になっているため、

何度も読み返したくなり、毎号、永久保存版になるような雑誌です。


今回は美術館特集。


私も美術館が大好きで、子供のころ、家族で日本中の美術館に

旅行で行っていました。


今週も母と大山崎山荘美術館へ行ってきます（*^_^*）

なんか椅子とモネの絵があるみたいです。


また、京都市美術館ではフェルメールからのラブレター展をしています

（名前がいいですね）


フェルメールは何回みてもいい☆


美術館は癒されるしお勧めです☆

涼しいし（*^_^*）

でも並ぶほどぎゅうぎゅうなのはお勧めしません(*_*)

梅雨明け！

大林　由衣 (2011年7月11日 10:39) 0 0 票 0 票
梅雨明けましたね！

これから本格的な夏になりました（*^_^*）

お子さんとプールや海に行って楽しめる季節ですね☆

子供のころの夏休みの思い出って
すごく大切なものになって残りますね。

夏休みに家族で旅行に行ったり、
お父さんとカブトムシ採りに行ったり、
親戚やいとこと会えたり、

毎日ラジオ体操、プールなど

こんな思い出って、大人になって
戻りたいなとか
もう一度再現したいなー

とか感じます。


そこで、私も子供のころよく川へ行って遊んでたのが
懐かしくなって
昨日は川へ行きました。


子供のころ行っていた川は、初めはすごくきれいで、
魚がいっぱいいたのに
何年後かに行くと、バーベキュー客が来くるようになって
川が汚れ、魚がいなくなっていたのが
子供心にもの凄くショックでした。

もうそんなことにならないように、
大人になった今、私たちが気をつけて
いかなければと感じます。

ですので、当たり前ですが
しっかり掃除して帰りました(^_^)


それにしても川の水は冷たかったぁ
氷水みたいでした（**）
入るのにもの凄い勇気がいりました。
でも勇気出して、最後には飛び込みましたー
童心に帰れました(^_^)


川は涼しい☆避暑にはお勧めです☆

七夕☆

大林　由衣 (2011年7月 7日 13:04) 0 0 票 0 票
今日は七夕ですね（*^▼^*）

大阪では雨ですが...

みなさん短冊には願い事書きましたか？

子供のころは、毎年庭でササの飾り付け、
楽しみにしていました(^_^)


今年は会社で毎年恒例の
ササ飾りに短冊も☆

連日の雨でボロボロになってしまって
短冊の文字読めなくなりましたが、
みんななんて書いたんだろ...(^_^)

私は「与えられる前に与える人になれますように」
と書きました



そして今日は友達の誕生日でもあります。
7月7日生まれの彼女の名前は「ナナちゃん」
なんか、覚えてもらえるし、
誕生日が名前って、
生まれたことをずっと大切にしてもらってる感じがして
いいですね（＾ｖ＾）



今日のニュースで、
七夕の本日の午後8時?10時
節電をかけて全国で一斉消灯するとのこと。

節電も全国運動も素敵ですが、
七夕の夜にやるって、すごくロマンチックで
いいアイデアですね。

天気がいいところは、星がきれいに見えるんじゃないかな☆

これからもこの一斉消灯は定期的に行っていくみたいです。

早く帰って、星を眺め、自然に感謝しつつ
疲れを取る日。
そんな日がこれからも増えたらいいですね（*^_^*）

暑いですね

大林　由衣 (2011年7月 4日 13:29) 0 0 票 0 票
暑いですね(>_<)

事務所のエアコンが壊れたまま、
あっと言う間に7月になりました。

先日業務用の（現場用）大きい扇風機を
営業部の部屋に置いてもらって、
いくらかましになりましたが(^_^;)

外のお仕事の方は本当に大変だろなと思います。
工事現場の方など、くらくらしながら仕事してるそうです。

倒れないように気をつけて下さい(>_<)



最近地下鉄の駅などで「節電中」という張り紙を見ます。

「エアコンをひかえています」とのこと。

でも全然快適です。
電車の中は寒いくらい。

今までどれだけ贅沢に電気を使っていたのかと
感じました。



今回は震災から、このような全国的な節電につながっていますが、

本当はこれくらい当たり前にしてないといけないことですよね。
co2　25％削減と掲げながら、今までこんな全国的な節電活動なんて
無かったですよね。

反省です。


もっとつねに危機感をもたないと。


私も家でできることを考えて...

暑いときは、エアコンをつける前に、扇風機で耐える。とか、

最近ハマっているのはかき氷（*^_^*）
かき氷機で作れば、安くてひんやり☆

休日は川などの避暑地に行こうかなと考えています。


本当は一日中水につかっていたいけど...(>_<)





さぁ、今週も暑さに負けず、がんばりましょう！
