現場見学しました

大林　由衣 (2011年8月 3日 20:04) 0 0 票 0 票

昨日は、進和建設工業が

建設中のマンションの

現場見学に行きました。

 

いつも完成見学会は開いていますが、

工事途中を見る機会は少ないので、

楽しみで、わくわくしながらいきました。

 

場所は大阪市道修町。

めちゃくちゃビジネス街です☆

ビジネス街で働きながら、

すぐに帰れる！

そんな便利な場所に

 

おっしゃれ~なデザイナーズマンションを

つくっています。

 

工事中の中身では

一見どこがどうなるかわかりませんが、

きっとここがああなって、こうなってって

想像するとわくわくします。

 

完成が楽しみです☆

現場には完成パースの看板がかかっていますよ

 

こんなマンションになりますよ~(#^.^#)
