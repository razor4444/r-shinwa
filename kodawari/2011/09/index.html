大林　由衣 (2011年9月29日 16:53) 0 0 票 0 票

光熱費ゼロの家

「casa sole カーサ・ソーレ」

のキャンペーンが始まります！！

 

 

 

期間は

2011年9月23日(金)　→　2011年12月10日(土)

 

 

どんなキャンペーンかというと、

「日本でいちばんエコな家」のカーサソーレに

さらに700万円分の性能を無料でお付けする

というキャンペーンです！

 

期間中に応募され選で当選された方に、

プラスエナジーハウスバージョン費用として

700万円分を無料でお付けします！

 

※700万円分の負担金額は、casa sole1を基準に計算しています。

casa sole1以外の場合で700万円以上の金額が必要になる場合は、

お客様にてご負担いただきます。また、建築場所(準防火地域等)によっては

、行政指導に より本仕様通りに施工できない場合があります。

その際、指導内容によっては追加料金が発生する場合がございます。

 

 

詳しくは下記HPをご覧ください☆

http://www.casasole.jp/700/

キャンペーンの当選発表は

2011年12月22日(木)

Web公開抽選会を予定しています。
詳細は、登録頂いたメールアドレスにご連絡致します。
当選者の発表は、casa sole公式ホームページにて行ないます。

 

casa sole をお考えの方、この機会に

日本一の性能を持ったパッシブハウスにしませんか？(*^_^*)/

 

詳しい内容をTVで放送します。 

BS　TwellV（トゥエルビ）にて放送☆

私のベランダ☆

 

ちょっとバカンス風です(*^_^*)

 

癒される?

 

冬場は植物たちを家の中に入れていますが、

夏場は、外に出して、日を浴びてもらっています、

 

外に出したら元気になります！

 

 

そこの足元に

 

小さな鉢植えが...

 

 

そこには、ツルが一本...

 

 

 

なんとこれ、何回か前にブログにアップしてた

 

流しから生えてきたメロンです！！

 

 

土に植えたら、こんなに伸びました！

 

植物が育っていくのを見ていると、

なんだか嬉しいな(*^_^*)

 

もっともっと家じゅうを植物だらけにしたいです。

 

でも日当たりがあまりよくないので、

なかなか家の中では元気にそだちません...

 

やっぱり家は日当たりが大事ですよ！(*^。^*)

 

それは人間も。

 

人間は日光にあたらないと、

ビタミンＤが作られないらしいですよ。

 

ビタミンＤが不足すると、だるくなったり

倦怠感を感じたりするそうです。

 

家を建てる際は、明るいお部屋で

健康に過ごしましょう(*^^)v

デザイナーズマンションの見学会を

女性向けのマンション

外観と、白い床がかっこいい室内。
