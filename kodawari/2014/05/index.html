2014年5月アーカイブ

言葉の力

進和ホーム (2014年5月 7日 18:27) 0 0 票 0 票
新入社員として進和ホームに入ってから、一ヶ月が経ちました。
私の顔と名前を覚えて下さっている会社まわりの方が
少しずつ増えてきました。
大変喜ばしいことです！

今日も外に出てご挨拶の訪問をしていたのですが、
その中で二回目の訪問に関わらず、
『仕事熱心にご苦労様、あなたの会社は必ず繁盛するよ！』
と、ある一人の方からお言葉を頂戴いたしました。
本当に心の底から感情が湧き出てくるのを実感し、
たった一言の言葉でここまで人を動かせるのかと
改めて言葉の力と感謝の大切さを痛感しました。
これからも一人でも多くの方へ、私から『感動』を
与えていくよう精進してまいります！
　　　　　　　　　　　　　　　　　　　　　　　　　　佐野
