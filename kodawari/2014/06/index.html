2014年6月アーカイブ

『人生を謳歌する住まい』とは...

進和ホーム (2014年6月 2日 10:10) 0 0 票 0 票
進和ホームの家づくりには、 『人生を謳歌する住まい』という コンセプトが有ります。 趣味、ペット、習い事、仕事等、 こだわりの詳細を、 とことん納得がいくまで 話し合うことから始まります。 建築会社の都合や、予算の都合で、 こだわりの家づくりを諦めて 妥協の家づくりをすることは、 それぞれのご家族の人生に 大きな影響を及ぼします。 家づくりで、何より大切なことは 愛する家族が世界で唯一の幸せな ライフスタイルを送ることが出来る ステージをつくりあげることです。 これだけはこだわったという 家族の絆がある家は、 家族の成長や住まい方に応じて、 家自体の魅力を増していきます。 そこには、家族の絆を象徴する ストーリーがあるのです。 『人生を謳歌する住まい』を つくりあげるお手伝いが出来るよう 精一杯頑張っていきます。 　　　　　　　　　　　　　　　中村
