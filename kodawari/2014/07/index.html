イベント予告

進和ホーム (2014年7月22日 21:46) 0 0 票 0 票
来月のイベント予告を行います。 ８月度は、『家づくりカフェ』を行います。 家づくりカフェとは、どんなイベントなんでしょうか？ 本来、家づくりは、楽しいものです。 限られた予算で、限られた敷地に、限られた仕様で、家づくりをしても、おもしろくないですよね。 テレビで見るリフォーム番組やお宅訪問のコーナーは、すごく楽しくて、私もついつい見入ってしまいます。 人それぞれ、家族それぞれ、背景があって、その背景に応じた家づくりが出来たとき、本来の家づくりの楽しさを感じることが出来るのです。 そんな観点で、家づくりについて、楽しく話し合う場が、『家づくりカフェ』なのです。 現実はさておき、家づくりについて、楽しくお話ししませんか？ 日時：平成２６年８月９日（土） 　　　１４：００?１６：００ 場所：進和建設工業株式会社 　　　本社１Ｆギャラリー 住所：大阪府堺市北区百舌鳥梅町１?３０?１ 電話：０７２?２５９?０６７２ 担当：中村・佐野

ありがとうございました

進和ホーム (2014年7月14日 08:59) 0 0 票 0 票
今月は、『七夕夢大賞』と『家族が仲良く暮らすためのセミナー』の２つのイベントを進和ホーム株式会社本社にて開催させていただきました。 天候が優れぬなか、ご参加いただきましたお客様、本当にありがとうございました。 進和ホームでは、このようなイベントを、継続的に行うことで、地域の皆様に楽しんでいただき、地域貢献していきたいと考えています。 皆様の笑顔がたくさん見られますように、積極的にイベント企画と情報の発信をおこなって参りますので、今後とも、よろしくおねがいいたします。 中村

七夕夢大賞

進和ホーム (2014年7月 5日 17:49) 0 0 票 1 票
　本日、弊社玄関先に、笹が二本設置されました。 　社員全員に短冊が配布され、みんなにこにこしながら、願い事を記入しています。 　進和グループでは、社員だけでなく、地元の皆様にも、この笹を利用し、願い事を叶えて頂きたきたいと考えております。 　つきましては、７月７日（月）１５時?１９時、弊社玄関前にて、七夕夢大賞というイベントを開催いたします。 　イベント開催中に、短冊に願い事をご記入の上、弊社玄関前の笹に取り付けて頂いた方に、参加賞（先着５０名様に「おかしセット」）のプレゼントがございます。 　また、素敵なお願い事をされた方３名に、たいしょう、おりひめ賞、ひこぼし賞がございます。 　皆様お誘いあわせの上、奮ってご参加ください。 中村
