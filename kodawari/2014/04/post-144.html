人のために尽くす

進和ホーム (2014年4月24日 18:35) 0 0 票 0 票
このタイトルは進和建設工業の
大切にしている精神です。
また今日一日、外を回って話を
伺うと、自分がどれだけド真剣に
相手のことを考えられているか、
自分を犠牲にしてまでも相手を
考えた行動が取れているか
ということを自問自答しました。
やはり、まだまだ不十分な点が
見つかりました。
仕事だけでなく、人生も成功するために
自分自身の姿勢を再度見つめ直し
住宅を通じて、人を幸せにしていくために
日々精進していきます！
